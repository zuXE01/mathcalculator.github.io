Fraction Calculator

Square Calculator

X²

This calculator computes the square of any number you enter. Squaring a number means multiplying it by itself. Simply input the number and click **Calculate Square** to get the result.

Percentage Change Calculator

This calculator determines the percentage change from an original value (**V¹**) to a new value (**V²**). The formula used is: **(V² - V¹) / V¹ × 100** Enter the original and new values to calculate the percentage increase or decrease.

Perfect Cubes - The First 100 Perfect Cubes

A cube of a number is the result of multiplying the number by itself three times. This calculator computes the cube of any number you enter. Input a number and click **Calculate Cube** to get the result.
